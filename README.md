## Installing the Project
### Make sure you have Docker and Git installed on your pc
___
Clonning repository into your pc:
>git clone git@github.com:UlissesvictoR/admin.git

Now you just build the docker container:
> docker compose up

or:
> docker compose build

If you already build the container, you can use:
> docker compose up -d admin
